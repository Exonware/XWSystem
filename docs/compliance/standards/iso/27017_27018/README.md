# ISO/IEC 27017 & 27018 � Cloud Security and Privacy

**Applicability:** Cloud-hosted services, SaaS offerings, and data processing pipelines (mandatory from v2).  
**Purpose:** Extend ISMS to address cloud-specific security (27017) and personal data protection (27018).

---

## Key Requirements

- Shared responsibility model documentation.  
- Cloud service configuration hardening.  
- Customer data lifecycle management (collection, use, deletion).  
- Incident response and breach notification procedures.  
- Privacy impact assessments for personal data processing.
